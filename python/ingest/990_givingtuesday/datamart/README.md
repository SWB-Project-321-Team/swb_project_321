# GivingTuesday Data Mart data

Scripts and references for **GivingTuesday Data Marts** (bulk pre-built 990 extracts), as distinct from the per-EIN API in `../api/`.

- **Data Marts:** [990 Data Marts](https://990data.givingtuesday.org/datamarts/) — research-ready CSV extracts by form type, category (e.g. Geography, Basic Fields). Download and store under `01_data/raw/givingtuesday_990/datamarts/`, then filter to the EIN list in staging.
- **See:** `docs/990/990_data_fetch_plan.md` (Option B, Data Marts).

Add fetch or filter scripts here as needed.
