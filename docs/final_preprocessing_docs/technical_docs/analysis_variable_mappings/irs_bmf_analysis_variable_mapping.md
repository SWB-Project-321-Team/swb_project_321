This file documents the official IRS EO BMF analysis-ready variables.
All paths are repo-relative and the package is intentionally source-direct to IRS EO BMF.

## Analysis Data Dictionary

This file is the analysis-ready data dictionary for the IRS EO BMF analysis dataset. It lists each canonical analysis variable, whether the variable is direct, calculated, enriched, proxy, imputed, or unavailable, the source rule used to populate it, the provenance column when applicable, and source-specific limitations.

|canonical_variable|role|analysis_requirement|source_rule|provenance_column|notes|
|---|---|---|---|---|---|
|analysis_total_revenue_amount|direct|Total revenue|Direct IRS EO BMF REVENUE_AMT field from the filtered benchmark rows|analysis_total_revenue_amount_source_column|Source-direct IRS EO BMF amount retained as nullable numeric.|
|analysis_total_assets_amount|direct|Total assets|Direct IRS EO BMF ASSET_AMT field from the filtered benchmark rows|analysis_total_assets_amount_source_column|Source-direct IRS EO BMF amount retained as nullable numeric.|
|analysis_total_income_amount|direct|Total income|Direct IRS EO BMF INCOME_AMT field from the filtered benchmark rows|analysis_total_income_amount_source_column|Source-direct IRS EO BMF amount retained as nullable numeric.|
|analysis_ntee_code|direct|NTEE filed classification code|Direct IRS EO BMF NTEE_CD field from the filtered benchmark rows|analysis_ntee_code_source_column|Source-direct IRS EO BMF classification retained without overwrite.|
|analysis_subsection_code|direct|Subsection code|Direct IRS EO BMF SUBSECTION field from the filtered benchmark rows|analysis_subsection_code_source_column|Source-direct IRS EO BMF subsection retained without overwrite.|
|analysis_ntee_code_fallback_enriched|enriched|NTEE filed classification code|IRS EO BMF direct NTEE first, then NCCS BMF exact-year, then NCCS BMF nearest-year fallback|analysis_ntee_code_fallback_enriched_source_column|Analyst-facing resolved NTEE code with explicit IRS-first precedence.|
