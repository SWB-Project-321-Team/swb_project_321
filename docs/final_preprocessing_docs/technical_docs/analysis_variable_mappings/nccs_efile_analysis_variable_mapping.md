## Analysis Data Dictionary

This file is the analysis-ready data dictionary for the NCCS efile analysis dataset. It lists each canonical analysis variable, whether the variable is direct, calculated, enriched, proxy, imputed, or unavailable, the source rule used to populate it, the provenance column when applicable, and source-specific limitations.
