source package in this repo.

- This file is limited to the 990-family sources in scope for the current planning package:
  - GivingTuesday datamart

- External dependencies such as ACS, BLS, ERS, and USAspending are referenced in notes where they
  are required to complete a metric, but they are not documented here as primary deliverables.
- Current pipeline and staging column names are used as the primary exact field references.

## Current interpretation

- **BMF** is the primary source for census-style questions: organization counts, density, county assignment,
  NTEE field representation, and regionwide reference totals.
- **Postcard** is a small-filer supplement for `tax_year >= 2022`, not the default replacement for BMF.
- **GivingTuesday** and **NCCS efile** are the detailed filing sources for the limited-number-organization
  questions.
- **NCCS Core** now has its own source-specific analysis package, but it remains a secondary mapped
  source in this crosswalk because its current scope is `2022` only and it preserves overlapping
  `PC` / `PZ` rows by design.
